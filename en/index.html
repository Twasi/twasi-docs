Developer Documentation

Twasi is an open source Twitchbot. It is easy to use and the hosting is provided. It is built with a modular system.
There are three main Components to Twasi, the Twasi Core wich is, like the Name points out, the Core Application of the Twasi Twitchbot.
The Twasi Plugins provide the Functionality of the Chatbot and the Twasi Panel wich acts as an UI for Userfriendly access to all the Functions of Twasi.

Twasi Core

Twasi Core is the Core Application of the Twasi Twitchbot. It hosts all the other plugins, manages the connection to the interface (Twitch IRC) and manages data persistence.

Learn more

Twasi Plugins

Twasi itself does not provide many features to use in the Twitch chat. Instead it loads plugins that contain these features bundled in a .jar file.

Learn more

Twasi Panel

This is the management board for the Twitch Bot. You can control, enable, disable and manage plugins and Functions of the Bot here.

Learn more

Powerfull API

We provide a powerfull API to use for your personal Project.
The API enables you, the developer to use data from streams tracked by Twasi to create wonderfull features for Twasi members to use in their channels.
There are a lot of usefull endpoints to choose from to bring your idea to Twasi.

Find out more

Modular Structure

To make Twasi stand out from other similar Projects we introduced Twasi Plugins.With Twasi Plugins you can completely customize your personal instance of the Twasi Chatbot.Another benefit with of a modular Open Source Project is that other Developers can contribute to Twasi to make it even better.
